# Organismos

Referencia sobre cómo se organiza un comité distrital del Frente Amplio.

- **[Directrices para Comités de Base Distritales](./directrices-capb.md)** — la norma oficial que rige la conformación y funcionamiento de estos comités.
- **[Estructura Distrital](./estructura-distrital.md)** — Asamblea Distrital, Mesa de Coordinación y cómo se relacionan.
- **[Roles y Responsabilidades](./roles-y-responsabilidades.md)** — funciones de cada cargo de coordinación.
- **[Comisiones Temáticas](./comisiones-tematicas.md)** — las seis áreas de trabajo del distrito.

## Mapa orgánico

```
Asamblea Distrital
      │
      ▼
Mesa de Coordinación Distrital
      │
      ▼
Comisiones Temáticas (6)
```

Hacia arriba, el comité distrital se coordina con el Comité Ejecutivo Cantonal (CEC) y, a través de este, con las estructuras provinciales y nacionales del partido.
